# Контейнер верификации маркеров аутентификации
**Обозначение:** КВМА.00001-01  
**Версия:** 2.0  
**Разработчик:** Приходько А.Р., группа КП-23-17  
**Дисциплина:** Управление жизненным циклом информационных систем  
**РГУ нефти и газа (НИУ) имени И.М. Губкина, 2026**

Контейнер реализует трёхуровневую верификацию маркеров аутентификации для защиты от угрозы подмены субъекта сетевого доступа (УБИ.131 по классификации ФСТЭК России).

## Запуск
```bash
docker compose up --build
```
